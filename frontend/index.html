TradeSmart Stock Trading System

Plain HTML frontend with mock API integration.

Quick Start

  1. Go to Market to see stocks.
  2. Place a Buy/Sell order (it becomes PENDING).
  3. Use Execute Pending Orders to simulate execution.
  4. Check Portfolio and Transactions.
  5. Use Admin pages to create stocks and update market rules.
